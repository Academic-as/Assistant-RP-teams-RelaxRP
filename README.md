## 🚀 Как запустить

Откройте файл `index.html` в браузере — это стартовая страница приложения.
Нажмите кнопку **Открыть**, чтобы перейти к генератору RP-команд в `app.html`.

> Приложение работает полностью в браузере: ничего не нужно устанавливать.

👉 **[Открыть](https://academic-as.github.io/Assistant-RP-teams-RelaxRP/)**

## Описание

Это простая HTML-страница для быстрого выбора RP-шаблонов RelaxRP, заполнения нужных полей и копирования готовой команды.

## Особенности

- выбор категорий `SCP-239-RX` и `SCP-953`
- автоматическая генерация RP-команд
- динамическое заполнение текста
- превью итоговой команды
- кнопка `Копировать` для быстрого копирования
- кнопка `Стереть` для очистки полей одной карточки
- кнопка `Стереть всё` для полного сброса
