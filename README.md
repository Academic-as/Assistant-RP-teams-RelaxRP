## Описание

Это простая страница на HTML, которая помогает выбрать одну из готовых RP-команд, заполнить нужные поля и скопировать результат в буфер обмена.

## Особенности

- Выбор категорий SCP-239-RX и SCP-953
- Автоматическая генерация команд с заполнением текста
- Показывает превью итоговой команды
- Кнопка `Копировать` для быстрой отправки
- Кнопка `Стереть` для очистки полей одной карточки
- Кнопка `Стереть всё` для очистки всех вводов
